## 📌 Project Status

**Current Phase:** 🧪 Research & Architecture Design  

This project is in the investigation and planning stage.  
The objective is to design, document, and implement a fully functional home server using repurposed hardware.

---

## 🎯 Vision

DIY Home Server is an open initiative focused on:

- ♻️ Reusing old or unused devices  
- 🧠 Building AI-capable infrastructure  
- 🔐 Implementing secure-by-default systems  
- 📚 Documenting the entire process for replication  

The long-term goal is to transform consumer hardware (old laptops, phones, etc.) into a modular and scalable self-hosted environment.

---

## 🖥 Initial Hardware Setup

| Device | Purpose |
|--------|----------|
| Laptop (i7, 16GB RAM, RTX GPU) | Main server node |
| Android phone | Monitoring & remote control interface |

The architecture is designed to be hardware-agnostic so others can adapt it to their own devices.

## 📚 Why This Project Exists

Many people own unused hardware capable of running powerful services.

This project aims to:

- Demonstrate real-world systems engineering
- Encourage self-hosting and digital sovereignty
- Provide a structured learning path in DevOps & infrastructure
- Prove advanced computing environments can be built without enterprise budgets
